smokerLab is a personal BBQ/smoking recipe, how-to guide, and shopping list web app built for a Royal Gourmet CC1830S offset smoker.

**Stack:** Vite + React + TypeScript + Tailwind CSS 4
**Typography:** Bitter (display), Source Sans 3 (body), JetBrains Mono (code)
**Palette:** Custom smoke scale (warm grays, 50-950), ember (amber/orange), coal (deep red), ash (green for success/do's)
**Theme:** Dark default with light mode via data-theme attribute, persisted to localStorage
**Mascot:** "Missy" - stylized geometric dog face (border collie/cattle dog), orange collar, pink tag. 1024px source PNG, already converted to favicon sizes.
**Current logo:** SVG flame/ember shape used in nav and hero, not yet incorporating the dog mascot.
**Structure:** Pages (Home, Recipes, RecipeDetail, HowTos, HowToDetail, ShoppingList), markdown content system, category filters.

**Why:** Personal project for tracking smoking recipes and technique.
**How to apply:** Design suggestions should respect the warm, artisanal, slightly rugged tone. The app is content-focused, not flashy. Missy is a personality element, not cutesy branding.
